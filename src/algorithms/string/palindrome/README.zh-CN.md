# 回文检测

[回文](https://en.wikipedia.org/wiki/Palindrome) 是指正向和反向读起来相同的字符串。
回文字符串在表现上后半段刚好和前半段倒过来一样。

## 举例

以下是一些回文字符串的例子 (返回 `TRUE`):

```
- "a"
- "pop"     ->  p + o + p
- "deed"    ->  de + ed
- "kayak"   ->  ka + y + ak
- "racecar" ->  rac + e + car
```

以下不是回文字符串 (返回 `FALSE`):

```
- "rad"
- "dodo"
- "polo"
```
