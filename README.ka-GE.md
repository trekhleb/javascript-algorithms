## рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃћрЃЉрЃў

"рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ" рЃљрЃарЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА(рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА) рЃЮрЃарЃњрЃљрЃюрЃўрЃќрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃерЃћрЃюрЃљрЃ«рЃЋрЃўрЃА рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃњрЃќрЃљ рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃерЃў, рЃўрЃЏрЃњрЃЋрЃљрЃарЃљрЃЊ рЃарЃЮрЃЏ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃЮрЃЊрЃћрЃА рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃљрЃЊ рЃгрЃЋрЃЊрЃЮрЃЏрЃљ рЃЊрЃљ рЃЏрЃЮрЃЊрЃўрЃцрЃўрЃфрЃўрЃарЃћрЃЉрЃљ.
рЃљрЃюрЃБ, "рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ" рЃљрЃарЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА(рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА) рЃћрЃарЃЌрЃћрЃБрЃџрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃџрЃћрЃЦрЃфрЃўрЃљ, рЃЏрЃљрЃЌ рЃерЃЮрЃарЃўрЃА рЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃў, рЃЊрЃљ рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў, рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃў рЃљрЃдрЃюрЃўрЃерЃюрЃБрЃџ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃќрЃћ рЃАрЃљрЃЏрЃБрЃерЃљрЃЮрЃЊ.

рЃњрЃљрЃ«рЃАрЃЮрЃЋрЃЊрЃћрЃЌ, рЃарЃЮрЃЏ рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃА рЃљрЃЦрЃЋрЃА рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў рЃЊрЃљрЃЊрЃћрЃЉрЃўрЃЌрЃў рЃЊрЃљ рЃБрЃљрЃарЃДрЃЮрЃцрЃўрЃЌрЃў рЃЏрЃ«рЃљрЃарЃћрЃћрЃЉрЃў. рЃЏрЃћрЃбрЃў рЃДрЃБрЃарЃљрЃЊрЃдрЃћрЃЉрЃљ рЃБрЃюрЃЊрЃљ рЃЏрЃўрЃљрЃЦрЃфрЃўрЃЮрЃЌ рЃўрЃЏрЃљрЃА, рЃЌрЃБ рЃарЃљрЃбрЃЮрЃЏ рЃўрЃарЃЕрЃћрЃЋрЃЌ рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљрЃА, рЃЋрЃўрЃЊрЃарЃћ рЃўрЃЏрЃљрЃА, рЃЌрЃБ рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃюрЃЋрЃљрЃ«рЃЮрЃарЃфрЃўрЃћрЃџрЃЮрЃЌ(рЃЊрЃљрЃЋрЃљрЃўрЃЏрЃърЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃарЃЮрЃЌ) рЃўрЃњрЃў.

## рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃћрЃЉрЃў

рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃў рЃљрЃарЃўрЃА рЃфрЃљрЃџрЃАрЃљрЃ«рЃљ рЃњрЃќрЃљ(рЃгрЃћрЃАрЃћрЃЉрЃў) рЃўрЃЏрЃўрЃАрЃљ, рЃЌрЃБ рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃЊрЃљрЃЋрЃГрЃарЃљрЃЌ рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ. рЃћрЃА рЃљрЃарЃўрЃА
рЃгрЃћрЃАрЃћрЃЉрЃўрЃА рЃАрЃўрЃЏрЃарЃљрЃЋрЃџрЃћ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃќрЃБрЃАрЃбрЃљрЃЊ рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃЌрЃљрЃюрЃЏрЃўрЃЏрЃЊрЃћрЃЋрЃарЃЮрЃЉрЃљрЃА.(рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ рЃАрЃљрЃГрЃЏрЃџрЃўрЃА рЃЏрЃЮрЃЏрЃќрЃљрЃЊрЃћрЃЉрЃўрЃА рЃарЃћрЃфрЃћрЃърЃбрЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЕрЃљрЃўрЃЌрЃЋрЃљрЃџрЃЮрЃА рЃарЃЮрЃњрЃЮрЃарЃф рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃў)

`рЃЏ` - рЃЏрЃљрЃарЃбрЃўрЃЋрЃў, `рЃа` - рЃарЃЌрЃБрЃџрЃў

### рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃћрЃЉрЃў рЃЌрЃћрЃЏрЃћрЃЉрЃўрЃАрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ

- **рЃЏрЃљрЃЌрЃћрЃЏрЃљрЃбрЃўрЃЎрЃљ**
  - `рЃа` [рЃЉрЃўрЃбрЃћрЃЉрЃўрЃА рЃЏрЃљрЃюрЃўрЃърЃБрЃџрЃљрЃфрЃўрЃљ](src/algorithms/math/bits) - set/get/update/clear рЃЉрЃўрЃбрЃћрЃЉрЃў, рЃњрЃљрЃЏрЃарЃљрЃЋрЃџрЃћрЃЉрЃљ/рЃњрЃљрЃДрЃЮрЃцрЃљ рЃЮрЃарЃќрЃћ, рЃБрЃљрЃарЃДрЃЮрЃцрЃўрЃЌрЃљрЃЊ рЃњрЃљрЃЊрЃљрЃЦрЃфрЃћрЃЋрЃљ рЃЊрЃљ рЃАрЃ«рЃЋрЃљ.
  - `рЃа` [рЃЮрЃарЃЮрЃЉрЃўрЃЌрЃў рЃЏрЃфрЃБрЃарЃљрЃЋ рЃгрЃћрЃарЃбрЃўрЃџрЃўрЃљрЃюрЃў рЃарЃўрЃфрЃ«рЃЋрЃў](src/algorithms/math/binary-floating-point) - рЃЏрЃфрЃБрЃарЃљрЃЋрЃў рЃгрЃћрЃарЃбрЃўрЃџрЃўрЃљрЃюрЃў рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃўрЃА рЃЮрЃарЃЮрЃЉрЃўрЃЌрЃў рЃгрЃљрЃарЃЏрЃЮрЃЊрЃњрЃћрЃюрЃљ.
  - `рЃа` [рЃцрЃљрЃЦрЃбрЃЮрЃарЃўрЃљрЃџрЃў](src/algorithms/math/factorial)
  - `рЃа` [рЃцрЃўрЃЉрЃЮрЃюрЃљрЃЕрЃўрЃА рЃарЃўрЃфрЃ«рЃЋрЃў](src/algorithms/math/fibonacci) - рЃЎрЃџрЃљрЃАрЃўрЃЎрЃБрЃарЃў рЃЊрЃљ рЃЊрЃљрЃ«рЃБрЃарЃБрЃџрЃў рЃцрЃЮрЃарЃЏрЃўрЃА рЃЋрЃћрЃарЃАрЃўрЃћрЃЉрЃў
  - `рЃа` [рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃў](src/algorithms/math/prime-factors) - рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃўрЃА рЃърЃЮрЃЋрЃюрЃљ рЃЊрЃљ рЃЏрЃљрЃЌрЃў рЃЊрЃљрЃЌрЃЋрЃџрЃљ рЃ░рЃљрЃарЃЊрЃў-рЃарЃљрЃЏрЃљрЃюрЃБрЃ»рЃљрЃюрЃўрЃА рЃЌрЃћрЃЮрЃарЃћрЃЏрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ
  - `рЃа` [рЃЏрЃљрЃарЃбрЃўрЃЋрЃЮрЃЉрЃўрЃА рЃбрЃћрЃАрЃбрЃў](src/algorithms/math/primality-test) (рЃАрЃљрЃфрЃЊрЃћрЃџрЃў рЃњрЃљрЃДрЃЮрЃцрЃўрЃА рЃЏрЃћрЃЌрЃЮрЃЊрЃў)
  - `рЃа` [рЃћрЃЋрЃЎрЃџрЃўрЃЊрЃћрЃА рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃў](src/algorithms/math/euclidean-algorithm) - рЃБрЃЊрЃўрЃЊрЃћрЃАрЃў рЃАрЃљрЃћрЃарЃЌрЃЮ рЃњрЃљрЃЏрЃДрЃЮрЃцрЃўрЃА (рЃБрЃАрЃњ-GCD) рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃљ
  - `рЃа` [рЃБрЃЏрЃфрЃўрЃарЃћрЃАрЃў рЃАрЃљрЃћрЃарЃЌрЃЮ рЃ»рЃћрЃарЃљрЃЊрЃў](src/algorithms/math/least-common-multiple) (рЃБрЃАрЃ»-LCM)
  - `рЃа` [рЃћрЃарЃљрЃбрЃЮрЃАрЃЌрЃћрЃюрЃћрЃА рЃАрЃљрЃфрЃћрЃарЃў](src/algorithms/math/sieve-of-eratosthenes) - рЃДрЃЋрЃћрЃџрЃљ рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃарЃўрЃфрЃ«рЃЋрЃўрЃА рЃърЃЮрЃЋрЃюрЃљ рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃа рЃЏрЃЮрЃфрЃћрЃЏрЃБрЃџ рЃарЃўрЃфрЃ«рЃЋрЃљрЃЏрЃЊрЃћ
  - `рЃа` [рЃљрЃарЃўрЃА рЃЮрЃарЃўрЃА рЃ«рЃљрЃарЃўрЃАрЃ«рЃў](src/algorithms/math/is-power-of-two) - рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃљрЃарЃўрЃА рЃЌрЃБ рЃљрЃарЃљ рЃарЃўрЃфрЃ«рЃЋрЃў рЃЮрЃарЃўрЃА рЃ«рЃљрЃарЃўрЃАрЃ«рЃў
  - `рЃа` [рЃърЃљрЃАрЃЎрЃљрЃџрЃўрЃА рЃАрЃљрЃЏрЃЎрЃБрЃЌрЃ«рЃћрЃЊрЃў](src/algorithms/math/pascal-triangle)
  - `рЃа` [рЃЎрЃЮрЃЏрЃърЃџрЃћрЃЦрЃАрЃБрЃарЃў рЃарЃўрЃфрЃ«рЃЋрЃў](src/algorithms/math/complex-number) - рЃЎрЃЮрЃЏрЃърЃџрЃћрЃЦрЃАрЃБрЃарЃў рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃў рЃЊрЃљ рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃў рЃЏрЃљрЃЌрЃќрЃћ
  - `рЃа` [рЃарЃљрЃЊрЃўрЃљрЃюрЃў рЃЊрЃљ рЃњрЃарЃљрЃЊрЃБрЃАрЃў](src/algorithms/math/radian) - рЃарЃљрЃЊрЃўрЃљрЃюрЃўрЃЊрЃљрЃю рЃњрЃарЃљрЃЊрЃБрЃАрЃерЃў рЃњрЃљрЃЊрЃљрЃДрЃЋрЃљрЃюрЃљ рЃЊрЃљ рЃърЃўрЃарЃўрЃЦрЃўрЃЌ рЃњрЃарЃљрЃЊрЃБрЃАрЃўрЃЊрЃљрЃю рЃарЃљрЃЊрЃўрЃљрЃюрЃерЃў рЃњрЃљрЃЊрЃљрЃДрЃЋрЃљрЃюрЃљ
  - `рЃа` [рЃАрЃгрЃарЃљрЃцрЃў рЃ«рЃљрЃарЃўрЃАрЃ«рЃерЃў рЃљрЃДрЃЋрЃљрЃюрЃљ](src/algorithms/math/fast-powering)
  - `рЃа` [рЃ░рЃЮрЃарЃюрЃћрЃарЃўрЃА рЃЏрЃћрЃЌрЃЮрЃЊрЃў](src/algorithms/math/horner-method) - рЃърЃЮрЃџрЃўрЃюрЃЮрЃЏрЃўрЃА рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃљ
  - `рЃа` [рЃЏрЃљрЃбрЃарЃўрЃфрЃћрЃЉрЃў](src/algorithms/math/matrix) - рЃЏрЃљрЃбрЃарЃўрЃфрЃћрЃЉрЃў рЃЊрЃљ рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЏрЃљрЃбрЃарЃўрЃфрЃБрЃџрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃў (рЃњрЃљрЃЏрЃарЃљрЃЋрЃџрЃћрЃЉрЃљ, рЃбрЃарЃљрЃюрЃАрЃърЃЮрЃќрЃўрЃфрЃўрЃљ рЃЊрЃљ рЃАрЃ«рЃЋрЃљ)
  - `рЃа` [рЃћрЃЋрЃЎрЃџрЃўрЃЊрЃБрЃарЃў рЃЏрЃљрЃюрЃФрЃўрЃџрЃў](src/algorithms/math/euclidean-distance) - рЃЏрЃљрЃюрЃФрЃўрЃџрЃў рЃЮрЃа рЃгрЃћрЃарЃбрЃўрЃџрЃА/рЃЋрЃћрЃЦрЃбрЃЮрЃарЃА/рЃЏрЃљрЃбрЃарЃўрЃфрЃљрЃА рЃерЃЮрЃарЃўрЃА

### рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃћрЃЉрЃў рЃърЃљрЃарЃљрЃЊрЃўрЃњрЃЏрЃћрЃЉрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ

рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃБрЃџрЃў рЃърЃљрЃарЃљрЃЊрЃўрЃњрЃЏрЃљ рЃљрЃарЃўрЃА рЃњрЃћрЃюрЃћрЃарЃўрЃЎрЃБрЃџрЃў рЃЏрЃћрЃЌрЃЮрЃЊрЃў рЃљрЃю рЃЏрЃўрЃЊрЃњрЃЮрЃЏрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃћрЃцрЃБрЃФрЃюрЃћрЃЉрЃљ рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃћрЃЉрЃўрЃА рЃЎрЃџрЃљрЃАрЃўрЃА
рЃЊрЃўрЃќрЃљрЃўрЃюрЃА. рЃћрЃА рЃљрЃарЃўрЃА рЃљрЃЉрЃАрЃбрЃарЃљрЃЦрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃБрЃцрЃарЃЮ рЃЏрЃљрЃдрЃљрЃџрЃўрЃљ, рЃЋрЃўрЃЊрЃарЃћ рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃўрЃА рЃЎрЃЮрЃюрЃфрЃћрЃцрЃфрЃўрЃљ, рЃўрЃАрЃћрЃЋрЃћ рЃарЃЮрЃњрЃЮрЃарЃф
рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃў рЃљрЃарЃўрЃА рЃљрЃЉрЃАрЃбрЃарЃљрЃЦрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃБрЃцрЃарЃЮ рЃЏрЃљрЃдрЃљрЃџрЃўрЃљ, рЃЋрЃўрЃЊрЃарЃћ рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃБрЃџрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ.

**Troubleshooting**

рЃЌрЃБ linting рЃљрЃю testing рЃљрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА, рЃерЃћрЃћрЃфрЃљрЃЊрЃћрЃЌ рЃгрЃљрЃерЃљрЃџрЃЮрЃЌ `node_modules` рЃцрЃЮрЃџрЃЊрЃћрЃарЃў рЃЊрЃљ рЃ«рЃћрЃџрЃљрЃ«рЃџрЃљ рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃЮрЃЌ npm рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃў:

```bash
rm -rf ./node_modules

```

рЃљрЃАрЃћрЃЋрЃћ, рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ, рЃарЃЮрЃЏ рЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ рЃАрЃгрЃЮрЃа Node рЃЋрЃћрЃарЃАрЃўрЃљрЃА (`>=16`). рЃЌрЃБ рЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ [nvm](https://github.com/nvm-sh/nvm) Node рЃЋрЃћрЃарЃАрЃўрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃњрЃљрЃБрЃерЃЋрЃљрЃЌ `nvm use` рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА root рЃцрЃЮрЃџрЃЊрЃћрЃарЃўрЃЊрЃљрЃю рЃЊрЃљ рЃАрЃгрЃЮрЃарЃў рЃЋрЃћрЃарЃАрЃўрЃљ рЃљрЃўрЃарЃЕрЃћрЃЋрЃљ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ.

рЃЦрЃЋрЃћрЃЏрЃЮрЃЌ рЃЏрЃЮрЃфрЃћрЃЏрЃБрЃџ рЃЊрЃўрЃљрЃњрЃарЃљрЃЏрЃљрЃќрЃћ рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃўрЃърЃЮрЃЋрЃЮрЃЌ рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃњрЃљрЃЋрЃарЃфрЃћрЃџрЃћрЃЉрЃБрЃџрЃў рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃћрЃЉрЃўрЃА рЃќрЃарЃЊрЃўрЃА рЃарЃўрЃњрЃћрЃЉрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃБрЃџрЃўрЃљ Big O рЃљрЃдрЃюрЃўрЃерЃЋрЃюрЃўрЃЌ.
